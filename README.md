## Funcionalidades

- Remove e conclui tarefas
- Data atual das tarefas


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Natcardoso/todo_list.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
